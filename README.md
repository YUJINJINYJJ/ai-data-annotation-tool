# 数据标注工具集 v1.0.0 正式发布

## 🎯 项目简介
本工具集是面向AI数据标注场景的批量处理工具，首版核心功能为**JSON关键词批量匹配与达标校验工具**，解决大规模标注文件的关键词检索、统计、合规校验痛点，支持多线程加速、批量导出报告，完美适配标注数据集的快速质检需求。
## ✨ 功能特性

### 🔍 JSON 关键词匹配
- 递归搜索 JSON 文件中所有字段值
- 支持模糊匹配和阈值校验
- 可拆分数组为独立条目统计
- 导出完整的校验报告

### 🎬 视频标注验收
- 验证视频标注的帧数和目标数
- 支持自定义验收标准
- 自动匹配视频与验收标准
- 导出 CSV 验收报告

### 🎙️ 语音标注验收
- 验证语音标注的片段数、时长、标签
- 检查无声比例和其他声音比例
- 支持多种声音事件类型
- 批量处理多个 JSON 文件

### 📄 中文文本分析
- 批量提取 CSV/JSON 文件中的中文内容
- 可选关键词出现次数统计
- 汇总表格式输出

## ✨ 核心功能
1.  **双模式文件导入**：支持批量上传JSON文件 / 本地文件夹批量读取，适配百/千级文件批量处理
2.  **模糊关键词检索**：不区分大小写的模糊匹配，支持多关键词同时检索，自动兼容中文/英文逗号
3.  **灵活统计模式**：
    - 整体模式：整个JSON文件作为单条统计条目
    - 拆分数组模式：JSON数组内每个元素独立统计，适配批量标注数据集
4.  **自定义达标校验**：支持为每个关键词设置最低出现次数，自动生成达标/未达标报告与原因
5.  **多线程加速**：最高支持16线程并行处理，大幅提升大规模文件处理效率
6.  **完整报告导出**：一键导出3类CSV报告：
    - 完整校验报告（含全局统计+分文件详情）
    - 分文件匹配结果明细
    - 匹配值出现次数统计（降序）
7.  **异常文件识别**：自动识别无效JSON、读取失败文件，避免程序崩溃
